# Expression Weak — Anti-Patterns Détaillés

> Exemples d'expression de marque de niveau ❌ Faible et ⚠️ Moyen. Chaque exemple montre POURQUOI c'est faible et COMMENT le corriger.

---

## 1. Taglines Faibles — Analyse

### Pattern : Accueil vide

```
❌ "Bienvenue chez La Pause"
```

**Pourquoi c'est faible :**
- Aucune promesse — on accueille, et ensuite ?
- Interchangeable : "Bienvenue chez [n'importe quel nom]"
- Le visiteur n'apprend rien sur ce qui l'attend
- Score : Spécificité 1, Mémorabilité 1, Émotion 1, ADN 2 = **5/20**

**Comment corriger :**
- Identifier l'insight de la plateforme → Qu'est-ce qui rend ce lieu unique ?
- Formuler une promesse, pas un accueil

### Pattern : Passe-partout sectoriel

```
❌ "Votre restaurant au cœur de Narbonne"
```

**Pourquoi c'est faible :**
- "Au cœur de" = métaphore morte (pattern #8)
- Tout restaurant est quelque part — ce n'est pas distinctif
- Pourrait être n'importe quel restaurant de Narbonne
- Score : Spécificité 1, Mémorabilité 1, Émotion 1, ADN 1 = **4/20**

### Pattern : Énumération de qualités

```
❌ "Fraîcheur, saveur, convivialité"
```

**Pourquoi c'est faible :**
- Liste d'adjectifs/noms sans structure narrative (pattern #4 + #9)
- Pas de tension, pas de contraste
- N'importe quel restaurant promet fraîcheur et saveur
- Score : Spécificité 1, Mémorabilité 2, Émotion 1, ADN 2 = **6/20**

### Pattern : Promesse impossible à ne pas faire

```
❌ "Une cuisine de qualité, préparée avec passion"
```

**Pourquoi c'est faible :**
- Truisme — quel restaurant dit "cuisine médiocre, préparée sans passion" ?
- "Avec passion" = pattern #9 (valeur abstraite vidée de sens)
- Aucune tension, aucun ancrage contextuel
- Score : Spécificité 1, Mémorabilité 1, Émotion 1, ADN 2 = **5/20**

## 2. Taglines Moyennes — Pourquoi elles ne suffisent pas

### Début de spécificité, manque de tension

```
⚠️ "Une cuisine faite maison, avec amour"
```

**Ce qui fonctionne :** "Faite maison" = début de spécificité (vs. industriel)
**Ce qui manque :** "Avec amour" = cliché qui affaiblit la promesse. Pas de tension.
**Score :** Spécificité 3, Mémorabilité 2, Émotion 2, ADN 3 = **10/20**

**Comment passer à ✅ :** Remplacer le cliché par un contraste contextuel :
→ "Ce qui sort de la cuisine, c'est du vrai." (tension vrai/faux, ancrage concret)

### Factuel correct, pas mémorable

```
⚠️ "Brasserie, glacier, café — tout sous un même toit"
```

**Ce qui fonctionne :** Informatif, décrit l'offre
**Ce qui manque :** Aucune émotion, aucune image mentale, pas de promesse
**Score :** Spécificité 3, Mémorabilité 2, Émotion 1, ADN 2 = **8/20**

**Comment passer à ✅ :** Transformer le descriptif en promesse :
→ "Du café du matin à la glace de l'après-midi — une seule adresse, zéro hésitation."

### Émotion présente, pas spécifique

```
⚠️ "L'endroit où l'on se sent bien"
```

**Ce qui fonctionne :** Émotion (bien-être)
**Ce qui manque :** Universel — pourrait être un spa, un hôtel, un parc
**Score :** Spécificité 1, Mémorabilité 2, Émotion 3, ADN 2 = **8/20**

**Comment passer à ✅ :** Ancrer l'émotion dans le contexte spécifique :
→ "La galerie marchande s'arrête ici." (même émotion, mais ancrée)

## 3. Messages par Section Faibles

### Pattern : Même message reformulé

```
❌ Hero : "Bienvenue dans notre restaurant chaleureux"
❌ Carte : "Découvrez notre carte variée et gourmande"
❌ Contact : "N'hésitez pas à nous contacter"
```

**Pourquoi c'est faible :**
- 3 messages, 1 seule idée ("venez chez nous")
- Aucun angle distinct par section
- Tous les impératifs sont génériques (pattern #3)

**Version corrigée :**
```
✅ Hero : "La galerie marchande s'arrête ici." (Insight → contraste)
✅ Carte : "Ce qui sort de la cuisine, c'est du vrai." (Mission → authenticité)
✅ Contact : "Un appel, une table. C'est aussi simple que ça." (Promise → simplicité)
```

→ 3 angles distincts : contraste, authenticité, simplicité

### Pattern : Messages informatifs sans émotion

```
⚠️ Hero : "Restaurant La Pause — Brasserie, Glacier, Café"
⚠️ Carte : "Notre carte propose des plats variés pour tous les goûts"
⚠️ Contact : "Retrouvez-nous au centre commercial Narbonne"
```

**Pourquoi c'est insuffisant :**
- Informatif mais pas engageant
- Aucune promesse, aucune tension
- Le visiteur comprend ce que c'est, mais pas pourquoi il devrait s'en soucier

## 4. CTAs Faibles

### Tableau comparatif

| CTA ❌ | Pourquoi c'est faible | CTA ✅ | Pourquoi c'est mieux |
|--------|----------------------|--------|---------------------|
| "Contactez-nous" | Générique, pas de promesse | "Réservez votre table au 04 68..." | Spécifique, action claire |
| "En savoir plus" | Passif, vague | "Voir la carte du jour" | Concret, orienté valeur |
| "Découvrir" | Découvrir quoi ? | "Choisir mon menu" | Centré visiteur, actionnable |
| "Cliquez ici" | Technique, archaïque | "Appelez-nous maintenant" | Urgence + action humaine |
| "Nos services" | Descriptif, pas d'engagement | "Voir ce qu'on prépare aujourd'hui" | Vivant, temporel, curieux |

## 5. Ton — Exemples "À éviter"

### Restaurant galerie marchande

| Phrase ❌ | Pattern | Problème |
|----------|---------|----------|
| "Nous vous proposons une large gamme de plats" | Verbe faible + vague | "Large gamme" ne veut rien dire |
| "Venez profiter d'un moment agréable" | Impératif générique | "Agréable" est le minimum syndical |
| "Notre équipe passionnée vous accueille" | Auto-congratulation | Centré sur eux, pas sur le client |
| "Découvrez nos délicieuses spécialités" | Adjectif bouche-trou | "Délicieuses" n'ajoute rien |
| "Chez La Pause, la satisfaction client est notre priorité" | "Votre X est notre priorité" | Formule vide universelle |

### Serrurier

| Phrase ❌ | Pattern | Problème |
|----------|---------|----------|
| "Nous intervenons rapidement pour résoudre vos problèmes" | Passe-partout | Tout serrurier dit ça |
| "Faites confiance à notre expertise" | Injonction de confiance | La confiance se gagne, ne se décrète pas |
| "Votre sécurité est entre de bonnes mains" | Cliché + "entre de bonnes mains" | Métaphore morte |

## 6. Signal d'Alerte — Quand Recommencer

Si lors de la relecture, vous constatez **3+ éléments** correspondant aux patterns ci-dessus, le process créatif n'a pas été appliqué correctement. Actions :

1. **Revenir à la plateforme** — Relire l'Insight, l'Archétype, la Promise
2. **Reconstruire la Language Library** — Les mots manquent probablement de spécificité
3. **Relancer le process Diverger** — Cette fois avec les contraintes du Lot 3 et Lot 5
4. **Ne pas patcher** — Réécrire de zéro plutôt que d'essayer de sauver du copy faible
