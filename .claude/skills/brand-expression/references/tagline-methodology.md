# Tagline Methodology

> Référence détaillée pour la création de taglines, baselines et slogans de marque.

---

## 1. Brand Ladder

Le Brand Ladder est un outil de montée en abstraction. Il transforme un fait concret (feature) en bénéfice émotionnel, puis en valeur fondamentale. La tagline se situe au sommet — elle exprime la valeur, pas la feature.

### Les 4 niveaux

```
        ┌─────────────────┐
        │   CORE VALUE    │ ← Tagline vit ici
        │  (Valeur fond.) │
        └────────┬────────┘
        ┌────────▼────────┐
        │  EMOTIONAL      │ ← Baseline peut vivre ici
        │  BENEFIT        │
        └────────┬────────┘
        ┌────────▼────────┐
        │  FUNCTIONAL     │ ← USPs vivent ici
        │  BENEFIT        │
        └────────┬────────┘
        ┌────────▼────────┐
        │  FEATURE        │ ← Descriptions produit
        │  (Fait concret) │
        └─────────────────┘
```

### Exemple appliqué : Restaurant en galerie marchande

| Niveau | Contenu |
|--------|---------|
| **Feature** | Cuisine minute, service à table, produits frais |
| **Functional Benefit** | On mange bien, vite, dans un cadre agréable |
| **Emotional Benefit** | Un vrai moment de pause humaine dans le flux commercial |
| **Core Value** | Authenticité — le vrai au milieu de l'artificiel |

→ **Tagline** (Core Value) : "La galerie marchande s'arrête ici."
→ **Baseline** (Emotional) : "Un moment vrai, une cuisine vraie, au cœur de la galerie."
→ **USP** (Functional) : "Service à table, cuisine minute, produits frais."

### Exemple appliqué : Serrurier d'urgence

| Niveau | Contenu |
|--------|---------|
| **Feature** | Intervention 24/7, 30 min, toutes marques |
| **Functional Benefit** | Porte ouverte rapidement, à toute heure |
| **Emotional Benefit** | Soulagement, retour à la sécurité du foyer |
| **Core Value** | Protection — chaque porte protège quelqu'un |

→ **Tagline** : "Derrière chaque porte, quelqu'un qui compte."
→ **Baseline** : "Intervention d'urgence, 30 minutes, jour et nuit."

### Piège courant

Rester bloqué au niveau Feature ou Functional :
- ❌ "Cuisine faite maison avec des produits frais" (Feature)
- ❌ "Mangez bien, mangez vite" (Functional)
- ✅ "La galerie marchande s'arrête ici." (Core Value)

## 2. Language Library

La Language Library est une banque de mots construite AVANT la génération créative. Elle nourrit la divergence et empêche le vocabulaire générique.

### Construction

1. **Extraire** 20-30 mots depuis la plateforme de marque :
   - Depuis les Values : leurs noms + les mots de leurs définitions
   - Depuis l'Archétype : le vocabulaire typique de cet archétype
   - Depuis l'Insight : les mots qui décrivent la tension
   - Depuis les Proof Points : le vocabulaire factuel

2. **Classer** par registre :

| Registre | Description | Exemples (restaurant galerie) |
|----------|-------------|-------------------------------|
| Émotionnel | Mots de ressenti | pause, moment, vrai, chaleur, sourire |
| Factuel | Mots de preuve | minute, frais, table, service, maison |
| Sensoriel | Mots des 5 sens | goût, arôme, croquant, doux, chaud |
| Aspirationnel | Mots d'ambition | refuge, échappée, authenticité, âme |
| Contextuel | Mots du lieu/secteur | galerie, vitrine, comptoir, passage |

3. **Identifier les mots interdits** :
   - Mots génériques du secteur ("qualité", "passion", "savoir-faire")
   - Mots des concurrents (ceux qui apparaissent sur leurs sites)
   - Mots vides ("découvrir", "profiter", "offrir", "bienvenue")

### Utilisation

Pendant la phase de divergence, piocher dans la Language Library pour construire les variantes. Chaque variante doit utiliser au moins 1 mot de la library et 0 mot interdit.

## 3. Process de Génération : 25+ Variantes

### Pourquoi 25+ ?

La recherche en créativité montre que la qualité des idées augmente avec la quantité — les meilleures idées arrivent souvent après les 15 premières (effet de "second wind"). En pratique, 8 variantes minimum par élément, avec l'objectif d'en explorer 25+ pour la tagline principale.

### Structure de génération

Pour la tagline principale, générer par lots :

**Lot 1 — Les évidentes (5 variantes)**
Les premières idées qui viennent. Souvent génériques. Servent de "purge" pour libérer la créativité.

**Lot 2 — Les angles (5 variantes)**
Une par angle créatif : émotionnel, sensoriel, provocant, narratif, minimaliste.

**Lot 3 — Les contraintes (5 variantes)**
Imposer des règles : max 4 mots, commencer par un verbe, utiliser une métaphore, formuler en question, formuler en négatif.

**Lot 4 — Les combinaisons (5 variantes)**
Combiner des éléments des lots précédents. Prendre un angle + une contrainte.

**Lot 5 — Les audacieuses (5+ variantes)**
Les idées risquées : provocantes, poétiques, paradoxales, ultra-courtes. Pas d'autocensure.

### Tableau de suivi

```markdown
| Lot | # | Variante | Angle | Notes |
|-----|---|----------|-------|-------|
| 1 | 1 | "..." | Évidente | Purge |
| 1 | 2 | "..." | Évidente | — |
| ... | ... | ... | ... | ... |
| 5 | 25 | "..." | Audacieuse | Paradoxe |
```

## 4. Scoring et Sélection

### Grille de scoring

Appliquer la grille de l'étape 3 du process créatif (SKILL.md) aux 25+ variantes :

| Critère | Poids | Description |
|---------|-------|-------------|
| Spécificité | x1 | Non-interchangeable avec un concurrent |
| Mémorabilité | x1 | Reste en tête (rythme, image, surprise) |
| Émotion | x1 | Provoque un ressenti au-delà de l'information |
| Cohérence ADN | x1 | Alignée avec archétype + values + promise |

**Seuil** : Score ≥ 16/20 pour passer au tour suivant.

### Sélection en entonnoir

```
25+ variantes
    ↓ scoring (≥ 16/20)
~5-8 candidates
    ↓ test d'interchangeabilité
~3-4 finalistes
    ↓ test "Et alors ?"
~2-3 survivantes
    ↓ test d'archétype
1 tagline choisie
```

## 5. Raffinement Final

### Tagline → Baseline → Slogan

Les trois sont distincts :

| Élément | Rôle | Longueur | Exemple |
|---------|------|----------|---------|
| **Tagline** | Idée centrale, identité | 3-10 mots | "La galerie marchande s'arrête ici." |
| **Baseline** | Développe la tagline | 10-20 mots | "Un moment vrai, une cuisine vraie, au cœur de la galerie." |
| **Slogan** | Version rythmée/mémorable | 5-12 mots | "Ici, on prend le temps du vrai." |

### Techniques de raffinement

1. **Compression** : Supprimer chaque mot un par un — si le sens tient, le mot était superflu
2. **Rythme ternaire** : Tester des structures en 3 temps ("X, Y, Z" ou "X. Y. Z.")
3. **Allitération/Assonance** : Chercher des échos sonores naturels (pas forcés)
4. **Ponctuation** : Tester avec/sans point final, avec tiret, avec points de suspension
5. **Contraste** : Si la tagline joue sur un contraste, vérifier qu'il est lisible immédiatement

### Validation finale

- [ ] La tagline fonctionne seule (sans contexte)
- [ ] La baseline ajoute de l'information (ne reformule pas)
- [ ] Le slogan est distinct de la tagline (pas une variante)
- [ ] Les trois ensemble racontent une histoire cohérente
- [ ] Aucun mot interdit de la Language Library n'est présent
