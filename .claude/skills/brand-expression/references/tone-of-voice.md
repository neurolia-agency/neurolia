# Tone of Voice Methodology

> Référence détaillée pour la construction du système de ton de voix.

---

## 1. Spectrum Mapping

Le spectrum mapping positionne la marque sur des axes de communication. C'est la première étape — elle cadre tout le reste.

### Axes standards

Positionner la marque sur chaque axe (1-5) :

```
Formel    [1]——[2]——[3]——[4]——[5]  Informel
Sérieux   [1]——[2]——[3]——[4]——[5]  Ludique
Réservé   [1]——[2]——[3]——[4]——[5]  Enthousiaste
Technique [1]——[2]——[3]——[4]——[5]  Accessible
Distant   [1]——[2]——[3]——[4]——[5]  Intime
```

### Axes optionnels (selon secteur)

```
Classique [1]——[2]——[3]——[4]——[5]  Moderne
Sobre     [1]——[2]——[3]——[4]——[5]  Exubérant
Prudent   [1]——[2]——[3]——[4]——[5]  Audacieux
```

### Dérivation depuis l'archétype

Chaque archétype a un positionnement naturel sur ces axes :

| Archétype | Formel/Informel | Sérieux/Ludique | Réservé/Enthousiaste |
|-----------|-----------------|-----------------|----------------------|
| **Caregiver** | 2-3 | 2-3 | 3-4 |
| **Creator** | 2-3 | 3-4 | 3-4 |
| **Explorer** | 3-4 | 3-4 | 4-5 |
| **Hero** | 2-3 | 1-2 | 3-4 |
| **Innocent** | 3-4 | 3-4 | 4-5 |
| **Jester** | 4-5 | 5 | 5 |
| **Lover** | 2-3 | 2-3 | 3-4 |
| **Magician** | 2-3 | 2-3 | 3-4 |
| **Outlaw** | 4-5 | 3-4 | 4-5 |
| **Regular Guy** | 4-5 | 3-4 | 3-4 |
| **Ruler** | 1-2 | 1-2 | 2-3 |
| **Sage** | 2-3 | 1-2 | 2-3 |

> Ces positions sont des **tendances** — le contexte du client peut les ajuster. Un Caregiver en restauration familiale sera plus informel (3-4) qu'un Caregiver en clinique (1-2).

### Format de restitution

```markdown
## Positionnement Ton

| Axe | Position | Explication |
|-----|----------|-------------|
| Formalité | 3/5 — Décontracté professionnel | [Pourquoi] |
| Sérieux/Ludique | 3/5 — Chaleureux | [Pourquoi] |
| Réserve/Enthousiasme | 4/5 — Accueillant | [Pourquoi] |
| Technique/Accessible | 5/5 — Grand public | [Pourquoi] |
```

## 2. Construction du Registre

Le registre traduit le spectrum en règles d'écriture concrètes.

### Tutoiement / Vouvoiement

| Facteur | Tutoiement | Vouvoiement |
|---------|------------|-------------|
| Archétype | Jester, Regular Guy, Innocent, Explorer | Ruler, Sage, Hero, Caregiver (formel) |
| Cible | Jeune, communauté, lifestyle | Professionnels, seniors, B2B |
| Secteur | Tech, food casual, sport | Luxe, santé, juridique, immobilier |
| Marque | Challenger, disruptive | Établie, institutionnelle |

> **Règle workspace Neurolia** : Sauf indication contraire du client, les sites utilisent le **vouvoiement** (règle `CLAUDE.md` workspace).

### Niveau de formalité (1-5)

| Niveau | Description | Marqueurs |
|--------|-------------|-----------|
| 1 | Très formel | Phrases longues, passif, vocabulaire soutenu |
| 2 | Professionnel | Structures classiques, pas d'argot, courtois |
| 3 | Décontracté pro | Phrases directes, ton conversationnel, mais poli |
| 4 | Familier | Phrases courtes, expressions courantes, dynamique |
| 5 | Très familier | Argot assumé, interjections, ultra-court |

### Longueur de phrase type

| Formalité | Longueur moyenne | Exemple |
|-----------|-----------------|---------|
| 1-2 | 15-25 mots | "Nous vous accueillons dans un cadre chaleureux pour vous offrir une expérience culinaire authentique." |
| 3 | 8-15 mots | "Un vrai moment de pause, au cœur de la galerie." |
| 4-5 | 4-10 mots | "Du vrai. Du bon. C'est ici." |

## 3. Champ Lexical

Le champ lexical est la banque de vocabulaire opérationnelle de la marque.

### Construction

**Étape 1 — Extraire depuis la plateforme**

| Source | Mots à extraire |
|--------|-----------------|
| Values | Noms + implications concrètes |
| Archétype | Vocabulaire typique (voir table ci-dessous) |
| Insight/Tension | Mots des deux pôles du contraste |
| Proof Points | Vocabulaire factuel/concret |
| Secteur | Termes métier accessibles |

**Étape 2 — Vocabulaire par archétype**

| Archétype | Mots typiques |
|-----------|---------------|
| **Caregiver** | accueil, soin, attentif, protéger, accompagner, chaleur, confiance, rassurer |
| **Creator** | créer, unique, artisan, façonner, imaginer, original, atelier, savoir-faire |
| **Explorer** | découvrir, aventure, liberté, horizons, parcours, surprendre, territoire |
| **Hero** | défi, excellence, dépasser, force, engagement, performance, détermination |
| **Innocent** | simple, joie, bonheur, naturel, pur, frais, léger, sourire, harmonie |
| **Jester** | fun, rire, décalé, surprenant, léger, spontané, complice, pétillant |
| **Lover** | passion, désir, élégance, séduction, beau, raffiné, sensuel, intime |
| **Magician** | transformer, révéler, possible, vision, extraordinaire, métamorphose |
| **Outlaw** | briser, rebelle, différent, radical, audace, franc, sans compromis |
| **Regular Guy** | ensemble, quotidien, vrai, honnête, accessible, proche, concret, fiable |
| **Ruler** | maîtrise, contrôle, leader, prestige, autorité, excellence, premium |
| **Sage** | comprendre, savoir, expertise, analyse, profondeur, éclairé, précis |

**Étape 3 — Mots à Utiliser vs. Mots à Éviter**

| Catégorie | Critère de sélection |
|-----------|---------------------|
| **À utiliser** | Spécifique au client + cohérent archétype + évocateur |
| **À éviter** | Générique secteur + utilisé par concurrence + vide de sens |

Format de restitution :

```markdown
## Mots à Utiliser
> Chaque mot renforce le positionnement.
- **[mot]** — [pourquoi il est pertinent]
- **[mot]** — [pourquoi]

## Mots à Éviter
> Chaque mot affaiblit la singularité.
- **[mot]** — [pourquoi il est interdit]
- **[mot]** — [pourquoi]
```

## 4. Personnalité de Marque

### Les 3-4 traits

Chaque trait est un **adjectif** accompagné d'un exemple de phrase en contexte. L'exemple n'est pas optionnel — c'est ce qui rend le trait actionnable.

### Dérivation

Les traits découlent de :
1. L'archétype (1-2 traits)
2. Les values (1 trait)
3. L'insight/tension (1 trait qui crée la différenciation)

### Format

```markdown
## Personnalité

- **[Trait 1]** : "[Phrase que la marque dirait en situation X]"
  → Dérive de : Archétype [Nom]
- **[Trait 2]** : "[Phrase que la marque dirait en situation Y]"
  → Dérive de : Value [Nom]
- **[Trait 3]** : "[Phrase que la marque dirait en situation Z]"
  → Dérive de : Insight
```

### Exemple (restaurant galerie — archétype Caregiver)

```markdown
- **Chaleureux** : "Installez-vous, on s'occupe de tout."
  → Dérive de : Archétype Caregiver
- **Sincère** : "C'est simple : des produits frais, préparés devant vous."
  → Dérive de : Value Authenticité
- **Ancré** : "Depuis 10 ans, c'est ici qu'on se retrouve entre les vitrines."
  → Dérive de : Insight (vrai vs. artificiel)
```

## 5. Exemples de Phrases

### Méthode de production

Pour chaque trait de personnalité, produire des phrases dans des **situations réelles** du client :

| Situation | Description |
|-----------|-------------|
| Accueil client | Première impression (hero, page d'accueil) |
| Description offre | Présentation d'un plat/service |
| Incitation à l'action | CTA, invitation |
| Réassurance | Répondre à un frein/inquiétude |
| Réseaux sociaux | Post, story, légende |

### Format "Bien" / "À éviter"

```markdown
### Bien
- "[Phrase 1 — situation]" — Illustre [trait]
- "[Phrase 2 — situation]" — Illustre [trait]

### À éviter
- "[Phrase 1]" — [Pourquoi c'est un problème : générique / hors ton / etc.]
- "[Phrase 2]" — [Pourquoi]
```

### Règles

- Minimum 4 phrases "Bien", 3 phrases "À éviter"
- Chaque phrase "Bien" doit être utilisable telle quelle sur le site
- Chaque phrase "À éviter" doit avoir une justification spécifique (pas juste "trop générique")
- Les phrases doivent couvrir au moins 3 situations différentes
