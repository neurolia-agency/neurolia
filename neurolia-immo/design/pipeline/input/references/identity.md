## 1. Nom

| Cle | Valeur |
|-----|--------|

Le nom "Loc Immo" est utilise dans l'interface (sidebar, barre de navigation). Le mot "Dashboard" n'est pas affiche — c'est implicite puisque toute l'application est un tableau de bord.

---

## 2. Tagline

**"Tes locations, une seule vue."**

Alternatives testees :
- "Toutes tes resas, un seul ecran." (trop restrictif — l'app fait plus que les resas)
- "La gestion locative simplifiee." (trop generique)

La tagline apparait uniquement sur la page de connexion, sous le logo.

---

## 3. Positionnement

Dashboard Loc Immo est un **outil de gestion interne** pour la location courte duree. Ce n'est pas un produit SaaS public, ni un site marketing.

| Axe | Position |
|-----|----------|
| Type | Application web privee (dashboard) |
| Audience | Proprietaire + personnel de menage |
| Ton | Professionnel mais accessible |
| Visuel | Neutre, sobre — les donnees sont au premier plan |
| Complexite | Simple en surface, puissant en fonctionnalites |

---

## 4. Valeurs

### 4.1 Efficacite

L'application doit faire gagner du temps. Chaque ecran repond a une question precise : "Quelles resas aujourd'hui ?", "Quel menage faire ?", "Combien j'ai gagne ce mois-ci ?". Pas de clics inutiles, pas de decorations superflues.

### 4.2 Clarte

L'information est dense (calendrier multi-biens, listes de reservations, KPIs) mais toujours lisible. La hierarchie visuelle guide l'oeil. Les codes couleur sont coherents et significatifs.

### 4.3 Fiabilite

Zero reservation manquee. L'application est la source de verite unique. Quand une donnee s'affiche, elle est a jour. Les alertes signalent les anomalies avant qu'elles deviennent des problemes.

---

## 5. Personnalite

| Trait | Oui | Non |
|-------|-----|-----|
| Professionnelle | L'outil inspire confiance | Pas froide ni corporative |
| Accessible | Tutoiement, langage direct | Pas condescendante |
| Discrete | L'UI s'efface devant les donnees | Pas fade — les couleurs plateformes apportent de la vie |
| Reactive | Feedback immediat, mises a jour temps reel | Pas envahissante en notifications |

---

## 6. Logo et icone

### Approche recommandee

| Element | Recommandation |
|---------|---------------|
| Logo | Texte "Loc Immo" en font semi-bold, sans pictogramme |
| Icone (favicon, PWA) | Initiales "LI" dans un carre arrondi, fond neutre |
| Couleur logo | Texte en couleur `neutral-900` sur fond clair |

Le logo est volontairement minimal. C'est un outil de travail, pas une marque a promouvoir.
