# D01 Phase 2 : Plateforme de Marque

> **Brand & Identity** - Construction de la plateforme strategique.

## Objectif

Construire la plateforme de marque complete en utilisant trois frameworks complementaires : Brand Key (Unilever), Prisme de Kapferer, et Archetypes jungiens.

## Input

- Diagnostic de la Phase 1 (en memoire)
- `pipeline/input/imports/prd.md`
- `pipeline/input/assets/`

## Instructions

### 1. Brand Key (Unilever) - 9 Composantes

Remplir chaque composante :

```markdown
## Brand Key

### 1. Environnement Concurrentiel
Qui sont les alternatives directes et indirectes ?
- [App concurrente 1] : [positionnement]
- [App concurrente 2] : [positionnement]
- [Alternative non-digitale] : [comment font-ils sans app ?]

### 2. Cible
Qui utilise cette app ? (repris et enrichi du diagnostic)
- **Profil** : [age, contexte, digital literacy]
- **Moment d'usage** : [quand ouvrent-ils l'app]
- **Frequence** : [quotidien / hebdomadaire / ponctuel]

### 3. Insight
Quelle verite profonde motive l'usage ?
> "[Formulation de l'insight]"
- Tension : [ce qui frustre actuellement]
- Desir : [ce qu'ils voudraient idealement]

### 4. Benefices
Que gagne l'utilisateur ?
- **Fonctionnel** : [gain concret]
- **Emotionnel** : [ressenti]
- **Social** : [image projetee]

### 5. Valeurs et Croyances
Quelles convictions portent la marque ?
1. [Valeur 1] : [manifestation dans l'app]
2. [Valeur 2] : [manifestation dans l'app]
3. [Valeur 3] : [manifestation dans l'app]

### 6. Raisons de Croire
Pourquoi cette promesse est credible ?
- [Preuve 1]
- [Preuve 2]
- [Preuve 3]

### 7. Discriminateur
Quel est l'element unique et inimitable ?
> "[Une phrase]"

### 8. Essence de Marque
En 2-3 mots, l'ame de la marque :
> "[Essence]"

### 9. Personnalite
Si la marque etait une personne :
- **Apparence** : [description]
- **Comportement** : [comment elle agit]
- **Voix** : [comment elle parle]
```

### 2. Prisme de Kapferer - 6 Facettes

```markdown
## Prisme de Kapferer

| Facette | Cote | Description |
|---------|------|-------------|
| **Physique** | Emetteur | Signes visuels de la marque : [logo, couleurs, formes, icones app] |
| **Personnalite** | Emetteur | Traits de caractere : [3-5 adjectifs] |
| **Culture** | Emetteur | Systeme de valeurs : [references culturelles] |
| **Relation** | -- | Type de rapport avec l'utilisateur : [mentor / ami / expert / complice] |
| **Reflet** | Recepteur | Image de l'utilisateur type : [comment il se voit] |
| **Mentalisation** | Recepteur | Image interieure : [comment il se sent en utilisant l'app] |
```

### 3. Archetypes Jungiens

Identifier 1 archetype dominant + 1 secondaire parmi les 12 :

| Archetype | Motivation | Exemples apps |
|-----------|-----------|---------------|
| Innocent | Simplicite | Calm, Headspace |
| Sage | Comprehension | Wikipedia, Duolingo |
| Explorateur | Liberte | Airbnb, Maps |
| Heros | Maitrise | Nike Run Club, Strava |
| Rebelle | Revolution | Robinhood, Signal |
| Magicien | Transformation | Notion, Figma |
| Homme ordinaire | Appartenance | WhatsApp, Slack |
| Amant | Intimite | Spotify, Pinterest |
| Bouffon | Plaisir | TikTok, Duolingo |
| Soignant | Service | MyFitnessPal, Doctolib |
| Createur | Innovation | Canva, Procreate |
| Souverain | Controle | LinkedIn, Bloomberg |

```markdown
## Archetypes

### Dominant : [Archetype]
- **Motivation** : [motivation]
- **Manifestation dans l'app** :
  - UI : [comment ca se traduit visuellement]
  - Ton : [comment ca se traduit verbalement]
  - Interactions : [comment ca se traduit dans les gestes]

### Secondaire : [Archetype]
- **Motivation** : [motivation]
- **Role** : Nuance l'archetype principal pour [raison]
```

## Output

Creer `pipeline/output/01-brand/00-platform.md` contenant les 3 frameworks complets.
