# OpenClaw.ai

**URL** : https://openclaw.ai/
**Type** : Site SaaS / Dev tools (dark theme)
**Pertinence** : Direction artistique dark premium, palette, typographie

## Pourquoi cette référence

Ambiance dark premium qui sert de base à la direction artistique Neurolia. Fond ultra-sombre, accents vibrants, typographie display distinctive, espacements généreux. Le côté "tech moderne sans être générique" qu'on cherche.

## Ce qui nous intéresse

### Palette de couleurs
- Fond ultra-sombre : `#050810` avec couches `#0a0f1a`, `#111827`
- Contraste élevé texte/fond
- Accents vibrants sur fond sombre (pas gris fade)
- Double accent chaud/froid qui crée du dynamisme

### Typographie
- Font display distinctive (Clash Display) pour les titres
- Font corps lisible (Satoshi)
- Bonne respiration verticale (line-height 1.6)
- Antialiasing soigné

### Structure
- Espacements généreux sans être vide
- Bordures subtiles avec transparence (`rgba(136, 146, 176, .15)`)
- Layout clair, pas de surcharge
- Sections bien délimitées

### Détails
- Sélection de texte personnalisée (accent color)
- Glow subtils sur les accents (`rgba(0, 229, 204, .4)`)
- Transitions fluides, pas d'animations excessives
- Identité visuelle mémorable (mascotte)

## Adaptation Neurolia

| OpenClaw | Neurolia |
|----------|----------|
| Coral #ff4d4d + Cyan #00e5cc | Terracotta #C45C3B (accent unique, plus chaud) |
| Mascotte crustacé | Barre verticale 4px signature |
| CLI-forward | Business-forward |
| Hack culture | Premium accessible |

## Ce qu'on ne reprend pas

- Le côté "dev tools" trop technique
- La répétition de testimonials (carousel infini)
