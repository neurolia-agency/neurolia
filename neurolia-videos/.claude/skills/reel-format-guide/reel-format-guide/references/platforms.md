# Platform Specs

| Property | Value |
|---|---|
| Resolution | 1080 x 1920 px |
| Dead zone top | 0-150px |
| Dead zone bottom | 1700-1920px |
| Dead zone right | 900-1080px (interaction buttons) |
| Safe area | x:0-900, y:150-1700 |
| Duration | 15-180s (optimal: 15-60s) |
