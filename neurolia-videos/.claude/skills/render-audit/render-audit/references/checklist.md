# Visual Audit Checklist

Apply to each rendered frame.

## Composition
- [ ] No empty zone > 40% of screen without content
- [ ] Elements intentionally centered or offset
- [ ] Dot visible and logically positioned
- [ ] Content within Instagram safe zone (y:250-1580, x:65-1015)

## Color
- [ ] Dot is vibrant orange (#E8703A range), NOT brown or muted
- [ ] Text/background contrast ratio >= 4.5:1 (white on dark)
- [ ] Scene 1 pastels are bright and saturated (coral #FFD6E0, periwinkle #C3D4F7)
- [ ] No unexpected color artifacts or muddy tones
- [ ] Dark theme consistent across scenes 2-6 (#050810)

## Typography
- [ ] Titles >= 42px, readable on mobile
- [ ] Body text >= 24px
- [ ] No truncated or cropped words
- [ ] No vertically misaligned words (stagger artifact)
- [ ] Max 30% of screen area covered by text simultaneously

## Animation (deduce from frame sequence)
- [ ] No visual gap > 1 second without movement
- [ ] Squash/stretch visible at impacts
- [ ] Grow/shrink transition dramatic (scale >= 2.5x)
- [ ] Smooth settle after fast motion (no abrupt stop)

## 3D Effects
- [ ] Logo rotation visible (not flat) — check frames during rotation
- [ ] Service blocks enter with perspective tilt
- [ ] Dot renders as sphere (highlight + shadow visible, not flat circle)
- [ ] Scene transition tilt perceptible

## Continuity
- [ ] Dot position continuous between scenes (no teleportation)
- [ ] Background color consistent within each phase
- [ ] Logo/text don't coexist on same frame (Scene 3)

## Priority Classification
- **CRITIQUE**: Breaks visual quality or user comprehension
- **MOYEN**: Noticeable on close inspection, affects polish
- **MINEUR**: Nitpick, only visible in stills
