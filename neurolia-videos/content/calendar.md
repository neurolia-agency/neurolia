# Calendrier Editorial — Video Pipeline Neurolia Dashboard

## Strategie

### Positionnement
"Le dashboard tout-en-un pour agences web et freelances ambitieux."

### 5 piliers editoriaux

| Pilier | % | Types de videos |
|--------|---|-----------------|
| Product Showcase | 35% | Feature walkthroughs, module deep-dives, tours cinematiques |
| Problem-Solution | 25% | Avant/apres, "Arretez de faire X a la main" |
| Behind the Build | 20% | Code-to-UI animations, stack reveal, build logs |
| Tips & Value | 15% | Astuces gestion, KPI tracking, pipeline management |
| Social Proof | 5% | Chiffres cles, milestones, citations |

### Plateformes

| Plateforme | Ratio | Duree sweet spot | Frequence |
|------------|-------|-----------------|-----------|
| Instagram Reels | 9:16 | 15-30s | 3/sem |
| TikTok | 9:16 | 15-45s | 2/sem |
| YouTube Shorts | 9:16 | 30-60s | 2/sem |
| YouTube Long | 16:9 | 3-10min | 1/2 sem |

### Cross-posting
- FeatureFlash 9:16 → IG + TikTok + YT Shorts (meme fichier)
- DashboardTour 9:16 → IG + YT Shorts
- BeforeAfter 9:16 → IG + TikTok + YT Shorts
- ModuleDeepDive 16:9 → YouTube Long uniquement

## Voix IA

- **Outil** : ElevenLabs (10K chars/mois gratuit)
- **Voix** : Homme, francais, 30-40 ans, calme et confiant
- **Settings** : Stability 0.5, Similarity 0.75, Style 0.3, Multilingual v2
- **Fallback** : Edge TTS (Microsoft, gratuit illimite)

## Musique & SFX

- **Musique** : Pixabay Music (gratuit, sans attribution)
  - Tech corporate ambiant (~100 BPM)
  - Lo-fi chill pour deep dives
  - Upbeat pour feature flashes
- **SFX** : Whoosh (transitions), Click/Tap (UI), Bass Drop (stat reveals), Ding (success)
