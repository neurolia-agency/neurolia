# Tagline Methodology — Reference

## Brand Ladder

La Brand Ladder est l'outil de derivation tagline → plateforme. Chaque echelon monte en abstraction :

```
Attributs (faits concrets)
  → Benefices fonctionnels (ce que ca fait)
    → Benefices emotionnels (ce que ca fait ressentir)
      → Valeur terminale (pourquoi ca compte dans la vie)
```

### Application au process creatif

1. **Lister les attributs** depuis 00-platform.md > Proof Points
2. **Monter d'un echelon** : transformer chaque attribut en benefice fonctionnel
3. **Monter encore** : transformer chaque benefice en emotion
4. **Dernier echelon** : relier a une valeur universelle

**La tagline vit entre les echelons 2 et 3** — assez concrete pour etre specifique, assez emotionnelle pour resonner.

### Exemple applique

```
Attribut : "Ouvert 7j/7 de 6h a 22h" (fait)
  → Benefice fonctionnel : "Toujours la quand vous en avez besoin"
    → Benefice emotionnel : "La tranquillite de savoir qu'on peut compter dessus"
      → Valeur : Fiabilite, presence

Tagline candidate : "Toujours la." (echelon 2, concret)
Tagline candidate : "La certitude d'etre accompagne." (echelon 3, emotionnel)
```

## Variantes — Methodologie 8 angles

Chaque angle doit etre explore independamment. Ne pas "varier" une meme idee 8 fois — explorer 8 DIRECTIONS differentes.

### Verification de diversite

Apres generation des 8+ variantes :
- [ ] Au moins 3 longueurs differentes (courte 3-4 mots, moyenne 5-7, longue 8-10)
- [ ] Au moins 2 registres differents (affirmatif, interrogatif, imperatif, elliptique)
- [ ] Au moins 1 variante sensorielle (son, texture, gout, odeur)
- [ ] Au moins 1 variante par negation ("Pas de X, juste Y")
- [ ] Aucune variante ne commence par le meme mot qu'une autre

## Scoring — Grille detaillee

### Specificite (1-5)

| Score | Definition | Test |
|-------|-----------|------|
| 1 | Fonctionne pour n'importe quelle entreprise | Remplacer le nom → toujours valide |
| 2 | Fonctionne pour le secteur en general | Specifique au metier, pas a l'entreprise |
| 3 | Fonctionne pour 2-3 concurrents proches | Se differencie du generique |
| 4 | Difficile a attribuer a un autre | Reference implicite a un element unique |
| 5 | Impossible a voler | Ancre dans un fait, un lieu, une histoire propre |

### Memorabilite (1-5)

| Score | Test |
|-------|------|
| 1 | Oublie avant la fin de la lecture |
| 2 | Se souvient du sens, pas des mots |
| 3 | Se souvient de 2-3 mots cles |
| 4 | Se souvient de la phrase exacte apres 2 lectures |
| 5 | On la repete spontanement a quelqu'un |

### Emotion (1-5)

| Score | Reaction |
|-------|----------|
| 1 | Neutre, informatif, "ok" |
| 2 | Leger interet, acquiescement |
| 3 | Sourire, hochement de tete |
| 4 | "Ah oui, c'est vrai" — resonance personnelle |
| 5 | Reaction physique — sourire large, frisson, envie d'agir |

### Coherence ADN (1-5)

| Score | Alignement |
|-------|-----------|
| 1 | Deconnecte de la plateforme |
| 2 | Aligne sur 1 composant (ex: juste la promise) |
| 3 | Aligne sur 2 composants (ex: promise + archetype) |
| 4 | Incarne l'archetype + values + insight |
| 5 | Synthese parfaite — la plateforme condensee en une phrase |
