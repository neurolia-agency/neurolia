## Screenshots

Déposer les captures d'écran dans `references/screenshots/`. Nommer clairement :

```
screenshots/
├── ref1-hero.png              # Section précise qui te plaît
├── ref1-menu-hover.png        # Détail d'interaction
├── ref2-homepage-full.png     # Page entière si besoin
├── ref3-cards-services.png    # Pattern de cards intéressant
└── hors-ref-typo-inspi.png    # Inspiration typo trouvée ailleurs
```

**Règle de nommage** : `[ref ou source]-[section ou élément].png`

**Ce qui est le plus utile** : des screenshots de sections précises plutôt que des pages entières. "Le hero de ce site" est plus exploitable que "tout le site".

> Les screenshots sont analysés visuellement par Claude en A03.
> Ils servent à mesurer : proportions, espacements, couleurs, rythme, densité.
> L'URL dans sites.md donne le contexte ; le screenshot donne le visuel.
